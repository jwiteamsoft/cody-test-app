# cody-test-app

A simple Node.js application using Express.

## Features
- Single endpoint: `/`
- Response: `Hello from Cody`

## Requirements
- Node.js 18+ recommended
- npm

## Install
```bash
npm install
```

## Run
```bash
npm start
```

Then open:

```text
http://localhost:3000/
```

## Expected Response
```text
Hello from Cody
```
